# Curriculum Strategy

## Level Bands
- `beginner`: CEFR A1-A2
- `advanced`: CEFR B1-B2
- `proficiency`: CEFR C1-C2

## Curriculum Principles
- keep course structure deterministic
- let the LLM generate lesson details within strict boundaries
- prioritize intelligibility and communicative success
- review weak topics with spaced repetition

## Lesson Pattern
1. introduce lesson goal
2. model vocabulary or sentence pattern
3. prompt learner response
4. correct and explain briefly
5. request retry when needed
6. summarize and store result

## Exercise Types
- teach
- repeat
- roleplay
- grammar
- pronunciation
- conversation
- recap
